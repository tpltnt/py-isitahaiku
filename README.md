# py-isitahaiku
A simple abstraction for [isitahaiku.com](http://isitahaiku.com/) written in Python 3.

In action:
```
>>> from isitahaiku import isHaiku
>>> isHaiku("foo")
False
```
Test it by executing ```./test_isitahaiku.py```
